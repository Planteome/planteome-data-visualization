The size of the node is proportional to how many edges are linking to or from it. The color of the node is influenced by the P value calculated above. If there is a P value listed for the term in the table above, then the color is a shade of red - the more red, the higher the correlation (lower P value). The more gray, the less of a correlation there is. If the term is not in the table above (either no P value was calculated or it was not significant enough), the term is a blue node.
